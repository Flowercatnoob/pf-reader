【塑能】轻风(Breeze)[风]

等级：术士/法师/奥能师 0
施放时间：标准动作
成分：语言，姿势，器材（一个风车）
范围：近距 (25尺+5尺/2等级)
目标：一个生物
持续：一小时 (D)
豁免：意志，通过则无效 (无害)
抗力：有
目标周围吹起施法者选定风向的微风。
微风会让目标在对抗酷暑、酷热环境以及吐息攻击时的豁免，或是在对抗云雾类效果（如死云、臭云或吸入式毒物）和毒气的豁免上获得+2加值。该法术在真空环境或水下没有任何效果。
施法者只能维持一个该法术效果。在法术效果持续中再次施放该法术会导致前一个法术效果消失。

【咒法】(创造)淋湿(Drench)

等级：术士/法师/奥能师 0
施放时间：标准动作
成分：语言、姿势
范围：近距 (25尺+5尺/2等级)
目标：一个生物，或大型以下物体
持续：一轮
豁免：反射，通过则无效 (物体)
抗力：有 (物体)
一场突如其来的大雨将目标生物或物体整个淋湿。这场雨会在有效范围内追踪目标，持续给目标洒水。如果目标身上着火了，火焰会被浇灭。比营火要小的火焰（比如提灯或火把）会被法术效果下被浇灭。

【塑能】震颤(Jolt)[电]

等级：术士/法师/奥能师 0
施放时间：标准动作
成分：语言，姿势
范围：近距 (25尺+5尺/2等级)
目标：一个生物
持续：瞬间
豁免：无
抗力：有
施法者在远程接触攻击成功后对目标进行电击。造成1d3点电流伤害。

【塑能】疏影(Penumbra)[黑暗]

等级：术士/法师/奥能师 0
施放时间：标准动作
成分：语言，姿势，材料（少量煤）
范围：接触
目标：接触到的生物或物体
持续：10分钟/等级 (D)
豁免：意志，通过则无效 (无害)
抗力：有
目标被一层薄薄的阴影所笼罩。
如果目标拥有在明亮光线下会削弱或目盲的特性，那么在该法术效果持续时间内不会受到这些负面影响。
一个施法者只能维持一个该法术。在法术效果持续中再次施放该法术会导致前一个法术效果消失。

【变化】扎根(Root)[地]

等级：术士/法师/奥能师 0
施放时间：标准动作
成分：语言，姿势，材料（一把泥土）
范围：接触
目标：接触到的生物
持续：1分钟/等级 (D)
豁免：意志，通过则无效 (无害)
抗力：有
该法术会提高目标的下盘稳固性，增强对战技的防御能力。
目标在对抗使自身移动或倒地的战技时，CMD检定获得+2洞察加值。
并且在为了保持平衡，或者避免在地面上有沙、小石头等妨碍移动的环境中摔倒时的技能检定中获得+2表现加值。

【塑能】法师之勺(Scoop)[力场]

等级：术士/法师/奥能师 0
施放时间：标准动作
成分：语言、姿势
范围：近距 (25尺+5尺/2等级)
目标：直径6英寸的力场容器
持续：引导
豁免：无
抗力：无
施法者制造出一个小型的力场容器。
施法者可以用移动动作可以让容器在一轮内向任何方向移动最多15尺。施法者和容器之间的距离一旦超过有效范围，法术终止。
施法者可以用移动动作指挥容器倾斜以倒出其中液体。
容器可以盛载最多一品脱的液体，或是最多五磅的小型物体。
另外，如果施法者持续集中精神引导一分钟，可以用容器收集一品脱的液体，或是将内容物均匀的涂抹在物体表面。
